Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding tempature is in Fahrenheit. 


def cels_to_fahr(n)
  return (n.to_f * 9 / 5) + 32
end

print "Enter a Celsius value: "
c = gets.chomp

conversion = cels_to_fahr(c)
puts "The Celsius value #{c.to_s}C is 
equal to #{conversion.to_s} degrees Fahrenheit"



def get_ordinal_number(number)
  right_most_digit = number % 10
  
  if right_most_digit == 1 && !is_teen_exception?(number)
    number.to_s + 'st'
  elsif right_most_digit == 2 && !is_teen_exception?(number)
    number.to_s + 'nd'
  elsif right_most_digit == 3 && !is_teen_exception?(number)
    number.to_s + 'rd'
  else
    number.to_s + 'th'
  end
end

def is_teen_exception?(number)
  number % 100 == 11 ||  number % 100 == 12 || number % 100 == 13
end

print "Enter a number: "
number = gets.chomp.to_i
puts "That's the #{get_ordinal_number(number)} item"

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

			
print "How many items do you want to see?  "
items = gets.chomp.to_i

number = 1
while number <= items
  
  if number % 3 == 0 && number % 5 == 0
    puts "Foobar"
  elsif number % 3 == 0 
    puts "Foo"
  elsif number % 5 == 0
    puts "Bar"
  else
    puts number
  end
  
  number = number + 1
end
